## Installation

LWKBExtensions is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'LWKBExtensions'
```

**Carthage**
```ruby
github "luowei/LWKBExtensions"
```
